# 长白山野山参分布可视化
> **项目目标：** 通过高精度的 3D 区域建模与分布热力视图，直观呈现长白山野山参在山脉间的核心分布区域及其生境规律。

## 01 设计初衷
长白山野山参的分布具有极强的区域特征。本项目旨在利用工业设计的三维表现手段，将复杂的地理分布数据（如核心产区、坡向适宜度、森林覆盖率）转化为高精度的 3D 区域图。通过视觉化手段直观展示“黄金分布区”与地形特征、水系走向之间的深层联系，为生态研究与品牌叙事提供专业支撑。

## 02 分布精度和生长环境
- **区域空间研究：** 模拟长白山脉核心区域的横向跨度，重点标注野山参分布最为集中的核心林区，展现其从山麓向腹地延展的空间密度变化。

## 03 核心视觉表现
- **3D 剖面逻辑：** 采用三维建模的方式，结合信息可视化展示野山参分布的疏密程度，清晰标注主要分布县域与人参分布区的交界关系。
- **色彩策略：** 运用高对比度的商业渲染色调，既保留科学准确性，又具备极高的艺术观赏性。

## 04 表现形式
- **多维度视图：** 包含全景鸟瞰、细节特写及核心产区 3D 剖面切片及典型生境（如针阔混交林）的局部分布特写。
- **视觉风格：** 追求极简的地图美感与硬核的科学数据感，力求在复杂的地理信息与高端商业视觉表现之间达成平衡。

## 01 相关资料与调研 (Research)
> **设计依据：** 基于泰山地质构造的科学性还原。

- **地质背景：** 研究泰山岩群的演化历史，重点关注其特有的片麻岩（Gneiss）与花岗岩（Granite）层理结构。
- **结构参考：** 采集地理科普书籍中的剖面逻辑，确立地下土壤（棕壤、褐土）与岩石基底的过渡关系。
- **视觉参考：** 调研高端地理杂志的插画风格，确立“科学严谨+商业美感”的视觉基调。


## 02 建模与渲染 (Modeling & Rendering)
> **技术执行：** 从几何构建到质感还原的工业化流程。

- **高精度建模：** 使用 Rhino 进行地质分层建模，通过高精度网格控制（Mesh Control）确保剖面边缘的几何连续性。
- **环境搭建：** 在 C4D/Blender 中构建多光源照明系统，模拟自然光射入剖面的视觉效果。
- **材质表现：** 利用置换贴图（Displacement）与物理材质，还原岩石断裂面的真实粗糙质感与矿物纹理。


## 03 可视化表现 (Visualization)
> **最终成果：** 面向地理科普的 3D 视觉展示。

- **多维视图：** 包含核心三维切片图、俯瞰全景图及局部构造特写。
- **信息传达：** 通过色彩管理区分不同的地质年代与岩石类型，实现复杂信息的直观化传达。
- **输出规格：** 达到印刷级精度，可用于科普书籍、展厅看板或专业数字媒体展示。
